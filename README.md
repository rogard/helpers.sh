# script.sh

Unix shell scripts.

## Usage
```sh
  chmod +x sh/<module>.sh
  ./sh/<module>.sh [arguments]
```

## Module
### [clone](https://github.com/rogard/script.sh/blob/main/sh/clone.sh)

Clones a given file to a uniquely named file.

### [dnscrypt](https://github.com/rogard/script.sh/blob/main/sh/dnscrypt.sh)

Toggle DNS resolution through dnscrypt-proxy.
